NeuronPlot

NeuronPlot is a Python simulator for exploring neuronal excitability through phase-plane geometry: nullclines, equilibria, bifurcations, trajectories, separatrices, limit cycles, and time series.

View on GitHub Download Documentation

Phase-plane geometry

Visualize vector fields, nullclines, equilibria, separatrices, and global trajectory behavior.

Bifurcation analysis

Explore how neuronal behavior changes as external current and model parameters vary.

Educational simulator

Designed to make reduced neuron models geometrically intuitive without hiding the mathematics.
